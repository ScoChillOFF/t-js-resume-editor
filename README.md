# JS Resume editor
## Описание
Учебный проект по фронтенд-разработке. Приложение представляет собой редактор резюме с возможностью скачивания готового PDF-документа. В настоящее время есть только базовое редактирование текстовых полей.
## Запуск
Приложение доступно по [ссылке](https://scochilloff.github.io/t-js-resume-editor) \
Для локального запуска нужно ввести следующие команды:
```bash
npm ci
npm run dev
# Или
npm ci
npm run build
npm run preview
```
